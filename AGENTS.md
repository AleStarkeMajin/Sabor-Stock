# Project Instructions

## UI & Design Guidelines
- **Style Consistency**: Maintain the current "Zinc/Modern" aesthetic. Use `zinc` color scales for primary UI elements.
- **Space Management**: Keep the clean, high-density but readable layout. Use consistent padding and rounded corners (`rounded-lg`).
- **Components**: Continue using the established patterns for Dialogs, Tables, and Forms.
- **Visual Feedback**: Use subtle hover states (e.g., `hover:bg-zinc-800` for dark buttons, `hover:bg-zinc-50` for table rows).
- **Icons**: Use `lucide-react` icons consistently with a size of `w-4 h-4` for standard actions.
